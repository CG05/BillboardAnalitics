Data Collection

개요

정형 데이터 수집을 위해 Spotify의 API와 python API 중 하나인 billboard.py 를 사용했다.
총 61년의 billboard chart top100차트를 수집해 json으로 저장 후 (곡 제목 - 아티스트) 형식으로
Spotify API에 검색하여 Spotify에 등재되어 있는지 확인하고 해당 곡의 music feature을 수집한다.

제목에 (x , X, Featuring, featuring, With, with, And, and) 이 포함되어 있으면 Spotify에서 검색되지 않아
raw string으로 변환해 검색했다. 그리고 라틴어가 포함되어 있는 경우에도 검색되지 않아
difflib.SequenceMatcher를 사용해 제목의 유사도를 비교해 50% 이상 유사할 경우 검색 결과를 반환했다.

>데이터 저장 형식

billboad.py에서 추출한 데이터 형식이다.
(json 으로 저장)
1 ~ 100위 곡의 List 안에 저장은 옆과 같다.
추출된 feature들의 종류는 총 10가지다.

>>Spotify에서 검색하기

billboard.py에서 추출한 json 데이터 안에서 중복된 곡들을 제거한 새로운 list를 생성한다.
제거된 song Set으로 Spotify에서 검색하기
전에 곡 제목을 raw string 변환
곡 제목을 소문자로 변경 후 유사도를 측정해
유사도 50%이상의 제목들만 저장한다.
